How to run the projects

Django:
while being in the django project dir (e.g. at /home/aiida/aiida-code/sqla_vs_dj/aiida_schema_exp/django/dj_json) run
./manage.py cmd_name

e.g.
./manage.py bulk_cr

SQLA:
The commands are in the SQLA project commands dir. You can run them as follows
python sqla_json/commands/bulk_add_to_group.py
